## Usage

```bash
$ mvn archetype:generate \
-DarchetypeGroupId=net.avdw \
-DarchetypeArtifactId=quickstart \
-DarchetypeVersion=1.0.0 \
-DgroupId=net.avdw \
-DartifactId=example \
-Dpackage=net.avdw.example \
-Dversion=1.0-SNAPSHOT
$ cd example
$ mvn package
$ java -jar target/example.jar
```
